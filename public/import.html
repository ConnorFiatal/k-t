CSV Import

Upload CSV files to bulk-import data. Download a template for each entity, fill it out, then upload. Existing records are skipped automatically.

Doors

Adds physical door entries. Existing doors with the same name are skipped.
name*, building, floor, room, description, door_type (KEYED/FOB/BOTH)*
Download template (doors-template.csv)

Keys

Imports individual keys. Key system is created if it doesn't exist. Parent key must already exist if specified.
key_number*, system_name*, level (GMK/MK/SUB_MASTER/CHANGE), parent_key_number, bitting, keyway, key_blank, notes
Download template (keys-template.csv)

Users (Staff)

Imports staff members. Duplicates matched by employee_id (if provided) or first + last name.
first_name*, last_name*, employee_id, department, title, email, phone, start_date, notes
Download template (users-template.csv)

Combinations (Key → Door)

Links keys to the doors they unlock. Import Doors and Keys first. Include system_name to disambiguate keys.
key_number*, system_name (recommended), door_name*
Download template (combinations-template.csv)

System Accounts

Imports system/application credentials. Duplicates matched by account_name + username pair.
account_name*, username*, password, url, access_level, notes
Download template (system-accounts-template.csv)
